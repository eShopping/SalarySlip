Step1: Open the solution in vs2012 or above version.

Step2: Set up start up project MvcSalarySlip project and run the application.

Step3: You will able to see a screen where you can add values like FirstName, LastName, per month, Gross Income and Super Rate.
             Notes: All field should required there is no validation is implemented on these textboxes.

Step4: After fill all the information click on generate button and you can see the desired result into table form. 

Step4: You can also run the test case with Test option> Run> All
VS testing environment will execute all the test case and pass the all test cases.

                             PayPeriod:  May 2018
